# User Stories: KenFinly Product Capabilities

This document outlines the user-centric requirements for KenFinly, bridging financial tracking, Stoic discipline, and gamification loops.

## 1. Halo Ritual & Daily Discipline
- **US001 (Golden Window Check-In):** As a practitioner of Stoic discipline, I want to execute my "Halo" ritual strictly between 04:00 AM and 10:00 AM, so that I can set my daily intent and lock in my full survival point rewards.
- **US002 (Late Entry Catch-up):** As a busy professional who missed the morning slot, I want the system to allow an optional "Late Entry" log in the afternoon, so that I can maintain my daily continuity habit tracking even if penalized with reduced points.
- **US003 (Streak Maintenance Loop):** As a highly gamified user, I want to see my continuous streak fire count increase every day I comply with the ritual, so that I am psychologically driven to protect my streak from burning down to 0.

## 2. Micro-Friction Financial Logging
- **US004 (One-Handed Mobile Entry):** As an urban user logging quick street-side or commute expenses, I want a persistent Floating Action Button (`+`) on the bottom-right that triggers a native-feeling Bottom Sheet, so that I can complete a money logging transaction in 3 seconds using only one thumb.
- **US005 (Immediate Fiscal Awareness):** As a busy founder, I want to instantly compare my dual gauge charts (Current Month vs. Previous Month) on the dashboard, so that I can visually measure my spending trajectory without parsing dense tabular accounting reports.

## 3. Credibility Tokens & Pledges (Halo Point Engine)
- **US006 (Promise Betting Leverage):** As an individual struggling with procrastination, I want to create a "Promise" (a strict behavior/spending commitment) and lock up a portion of my Halo Points (HP) as collateral, so that loss aversion forces me to achieve my goals.
- **US007 (Global Standings & Peer Comparison):** As a competitive individual, I want my verified discipline points to rank against other users on a Global Leaderboard, so that I can benchmark my personal credibility metrics transparently against the world.
